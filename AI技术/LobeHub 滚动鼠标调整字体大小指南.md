# LobeHub 滚动鼠标调整字体大小指南

## 概述

LobeHub 是一个现代化的 AI 应用平台，支持通过鼠标滚轮快速调整界面字体大小，提升个性化阅读体验。

## 使用方法

### 快捷键操作

| 操作 | Windows/Linux | macOS |
|------|--------------|-------|
| 放大字体 | `Ctrl` + 滚轮向上 | `Cmd (⌘)` + 滚轮向上 |
| 缩小字体 | `Ctrl` + 滚轮向下 | `Cmd (⌘)` + 滚轮向下 |

### 操作步骤

1. **打开 LobeHub** 应用或网页版
2. **按住修饰键**：
   - Windows/Linux：按住 `Ctrl` 键
   - macOS：按住 `Command (⌘)` 键
3. **滚动鼠标滚轮**：
   - 向上滚动 → 字体变大
   - 向下滚动 → 字体变小
4. **松开按键** 完成调整

## 适用场景

- 📱 **视力辅助**：为视力不佳的用户放大文字
- 🎯 **演示展示**：投屏时放大字体便于观众阅读
- 💻 **大屏优化**：在4K/高分辨率显示器上增大字体
- 📖 **舒适阅读**：根据个人阅读习惯调整大小

## 注意事项

### 缩放范围限制

- **最小值**：约 50% 原始大小
- **最大值**：约 200% 原始大小
- **默认值**：100%（可重置）

### 与浏览器缩放的区别

| 特性 | 鼠标滚轮缩放 | 浏览器缩放 (`Ctrl + +`) |
|------|-------------|------------------------|
| 影响范围 | 仅文字内容 | 整个页面（含UI元素） |
| 布局影响 | 保持原有布局 | 可能改变布局结构 |
| 精细度 | 更平滑的调整 | 固定比例步进 |

### 兼容性

- ✅ **LobeHub 桌面版**（Electron）
- ✅ **LobeHub 网页版**（基于浏览器）
- ⚠️ **部分旧版本**：可能需要更新至最新版本

## 重置字体大小

如果调整后想要恢复默认设置：

1. **快捷键重置**：
   - Windows/Linux: `Ctrl + 0`
   - macOS: `Cmd (⌘) + 0`

2. **设置菜单重置**：
   - 进入设置 → 外观 → 字体大小 → 重置为默认

## 常见问题

### Q: 鼠标滚轮无响应？

**A:** 检查以下几点：
- 确保按住了 `Ctrl` 或 `Cmd` 键
- 尝试刷新页面或重启应用
- 检查鼠标滚轮是否正常工作

### Q: 字体缩放后布局错乱？

**A:** 优先使用鼠标滚轮缩放（仅影响文字），而非浏览器整体缩放。

### Q: 如何保存我的字体大小偏好？

**A:** LobeHub 会自动保存你的字体设置，下次打开时会保持上次的大小。

## 相关技巧

### 配合触控板使用

- **macOS**：双指在触控板上滑动，同时按住 `Cmd` 键
- **Windows**：双指滑动，同时按住 `Ctrl` 键

### 快速切换预设大小

```
极小  →  小  →  默认  →  中  →  大  →  极大
```

可通过多次快速滚动在这些预设值间切换。
