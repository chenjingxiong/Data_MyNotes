# 

Claude Code 通过内置技能（Skills）可以高效操作 Microsoft Office 文档，包括 Word (.docx)、PowerPoint (.pptx) 和 Excel (.xlsx/.xlsm)。

## 📝 Word 文档操作 (docx)

使用 `docx` 技能可以处理 Word 文档的所有常见任务。

### 基本操作示例

```
# 创建新文档
请用 docx 技能创建一个新的 Word 文档，包含标题和段落

# 编辑现有文档
请读取 report.docx 并修改其中的第三章内容

# 处理修订和评论
请显示 document.docx 中的所有修订和评论
```

### 功能清单

| 功能 | 描述 |
|------|------|
| 创建文档 | 生成新的 .docx 文件，支持样式和格式 |
| 编辑内容 | 修改文本、段落、标题 |
| 修订追踪 | 查看和接受/拒绝修订 |
| 评论管理 | 添加、回复、删除评论 |
| 格式保留 | 编辑时保持原有格式 |

## 📊 Excel 电子表格操作 (xlsx)

使用 `xlsx` 技能处理电子表格数据。

### 基本操作示例

```
# 创建带公式的表格
请用 xlsx 技能创建一个包含 SUM 和 AVERAGE 公式的销售报表

# 分析数据
请分析 data.xlsx 中的销售数据并生成可视化图表

# 修改表格
请更新 budget.xlsx 中的 Q4 数据并重新计算所有公式
```

### 功能清单

| 功能 | 描述 |
|------|------|
| 创建表格 | 生成新的 .xlsx/.xlsm 文件 |
| 公式支持 | 支持所有 Excel 函数和公式 |
| 数据分析 | 分析数据并生成洞察 |
| 可视化 | 创建图表和数据透视表 |
| 格式化 | 应用条件格式、样式 |

## 🎯 PowerPoint 演示文稿操作 (pptx)

使用 `pptx` 技能创建和编辑幻灯片。

### 基本操作示例

```
# 创建演示文稿
请用 pptx 技能创建一个 10 页的产品介绍演示文稿

# 修改布局
请更改 presentation.pptx 的第三章幻灯片布局为两栏

# 添加备注
请为所有幻灯片添加演讲者备注
```

### 功能清单

| 功能 | 描述 |
|------|------|
| 创建演示 | 生成新的 .pptx 文件 |
| 布局管理 | 更改和应用幻灯片布局 |
| 添加内容 | 文本、图片、形状 |
| 演讲者备注 | 添加和编辑备注 |

## 💡 使用技巧

### 1. 明确指定技能
当提到 Office 操作时，明确指定使用哪个技能：
```
请用 docx 技能打开并编辑...
请用 xlsx 技能分析...
请用 pptx 技能创建...
```

### 2. 指定文件路径
对于编辑操作，提供文件路径：
```
请编辑 ~/Documents/report.docx
```

### 3. 批量操作
可以同时处理多个文件：
```
请分析 ~/Documents/Reports/ 文件夹下所有 .xlsx 文件的数据
```

### 4. 导出位置
Office 文档可以导出到指定位置：
- `~/Desktop` - 导出到桌面
- `~/Downloads` - 导出到下载文件夹
