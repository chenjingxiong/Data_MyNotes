## 按任务快速查找

**我要开始一个新功能**
→ `plan` → `tdd` → 框架 patterns → review → `verification-loop`

**我要修复一个 bug**
→ `go-build`/`kotlin-build` → review → 测试

**我要审查代码**
→ `{language}-review` → `security-review`（如适用）

**我要设计 API**
→ `api-design` → `backend-patterns`

**我要设计数据库**
→ `postgres-patterns` / `jpa-patterns` → `database-migrations`

**我要部署应用**
→ `docker-patterns` → `deployment-patterns`

**我要测试应用**
→ `{language}-testing` / `e2e-testing` → `verification-loop`
