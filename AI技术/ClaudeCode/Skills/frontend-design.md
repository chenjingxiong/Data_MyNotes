# Frontend Design

> [!tip|style-green] 🎨 Frontend Design
> **梯队**：效率倍增层 | **来源**：Anthropic 官方 | **状态**：✅ 已安装
>
> **一句话**：官方出品，告别"AI 味"设计

## 核心能力

- 生成高质量 HTML/CSS/React 代码
- 内建设计哲学系统（Tailwind + shadcn/ui）
- 避免常见的 AI 设计通病（千篇一律的渐变、间距混乱等）
- 支持 Figma 设计稿复刻

## 适用场景

- 快速原型开发
- Landing Page 创建
- Dashboard UI 构建
- 组件库生成
