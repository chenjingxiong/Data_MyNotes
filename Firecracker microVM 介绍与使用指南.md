## 1. д»Җд№ҲжҳҜ Firecrackerпјҹ

**Firecracker** жҳҜз”ұ **AWSпјҲAmazon Web Servicesпјү** ејҖеҸ‘зҡ„ејҖжәҗ**иҪ»йҮҸзә§иҷҡжӢҹжңәзӣ‘и§ҶеҷЁпјҲVMMпјү**пјҢеҹәдәҺ Linux **KVM** жһ„е»әпјҢдё“дёә**ж— жңҚеҠЎеҷЁпјҲServerlessпјү**е’Ң**е®№еҷЁзә§йҡ”зҰ»**еңәжҷҜи®ҫи®ЎгҖӮе®ғиғҪеңЁдёҚеҲ° **125ms** еҶ…еҗҜеҠЁдёҖдёӘе®Ңж•ҙзҡ„иҷҡжӢҹжңәпјҲmicroVMпјүпјҢжҸҗдҫӣдёҺдј з»ҹиҷҡжӢҹжңәеҗҢзӯүзҡ„е®үе…Ёйҡ”зҰ»пјҢеҗҢж—¶жӢҘжңүжҺҘиҝ‘е®№еҷЁзҡ„йҖҹеәҰе’Ңиө„жәҗејҖй”ҖгҖӮ

### ж ёеҝғзҗҶеҝө

```
дј з»ҹиҷҡжӢҹжңә:  еҠҹиғҪе®ҢеӨҮ вҶ’ йҮҚйҮҸзә§ вҶ’ еҗҜеҠЁж…ў (з§’зә§) вҶ’ ејәйҡ”зҰ»
е®№еҷЁ:       жңҖе°ҸеҢ–   вҶ’ иҪ»йҮҸзә§ вҶ’ еҗҜеҠЁеҝ« (жҜ«з§’зә§) вҶ’ е…ұдә«еҶ…ж ё
microVM:    жңҖе°ҸеҢ–   вҶ’ иҪ»йҮҸзә§ вҶ’ еҗҜеҠЁеҝ« (~125ms) вҶ’ ејәйҡ”зҰ» (KVM)
```

Firecracker зҡ„и®ҫи®Ўе“ІеӯҰжҳҜ **"жһҒз®Җи®ҫеӨҮжЁЎеһӢ"**вҖ”вҖ”еҸӘжЁЎжӢҹе·ҘдҪңиҙҹиҪҪзңҹжӯЈйңҖиҰҒзҡ„зЎ¬д»¶и®ҫеӨҮпјҲvirtio-netгҖҒvirtio-blockгҖҒдёІеҸЈжҺ§еҲ¶еҸ°пјүпјҢдёҚжЁЎжӢҹд»»дҪ•дј з»ҹйҒ—з•ҷи®ҫеӨҮпјҲж—  USBгҖҒж—  VGAгҖҒж—  ACPI иЎЁпјүпјҢд»ҺиҖҢеӨ§е№…зј©е°Ҹж”»еҮ»йқўе№¶еҠ йҖҹеҗҜеҠЁгҖӮ

> [!tip] дёәд»Җд№ҲйҖүжӢ© Firecrackerпјҹ
> еҰӮжһңдҪ йңҖиҰҒ **е®№еҷЁзҡ„йҖҹеәҰ** + **иҷҡжӢҹжңәзҡ„е®үе…Ёйҡ”зҰ»**пјҢFirecracker жҳҜзӣ®еүҚжңҖжҲҗзҶҹзҡ„ејҖжәҗж–№жЎҲгҖӮAWS Lambda е’Ң AWS Fargate зҡ„з”ҹдә§зҺҜеўғеқҮеҹәдәҺ Firecracker иҝҗиЎҢгҖӮ

## 2. ж ёеҝғзү№жҖ§

| зү№жҖ§ | иҜҙжҳҺ |
|---|---|
| **жһҒйҖҹеҗҜеҠЁ** | microVM еҗҜеҠЁж—¶й—ҙ < 125msпјҢд»Һеҝ«з…§жҒўеӨҚ < 1ms |
| **жһҒдҪҺејҖй”Җ** | жҜҸдёӘ microVM д»…йўқеӨ–еҚ з”Ё ~5MB еҶ…еӯҳ |
| **ејәе®үе…Ёйҡ”зҰ»** | еҹәдәҺ KVM зЎ¬д»¶иҷҡжӢҹеҢ–пјҢжҜҸдёӘ VM зӢ¬з«ӢеҶ…ж ёз©әй—ҙ |
| **жһҒз®Җи®ҫеӨҮжЁЎеһӢ** | д»…ж”ҜжҢҒ virtio-netгҖҒvirtio-blockгҖҒдёІеҸЈпјҢж”»еҮ»йқўжһҒе°Ҹ |
| **йҖҹзҺҮйҷҗеҲ¶** | еҶ…зҪ®зҪ‘з»ңеёҰе®ҪгҖҒIOPSгҖҒзЈҒзӣҳеҗһеҗҗйҮҸзҡ„зІҫз»ҶйҷҗжөҒ |
| **еҝ«з…§дёҺжҒўеӨҚ** | ж”ҜжҢҒеҲӣе»әеҶ…еӯҳеҝ«з…§ + зЈҒзӣҳеҝ«з…§пјҢеҸҜзһ¬й—ҙе…ӢйҡҶ/жҒўеӨҚ |
| **RESTful API** | йҖҡиҝҮ Unix Socket жҡҙйңІ JSON APIпјҢжҳ“дәҺйӣҶжҲҗ |
| **seccomp жІҷз®ұ** | VMM иҝӣзЁӢиҝҗиЎҢеңЁдёҘж јзҡ„ seccomp иҝҮж»ӨеҷЁдёӢ |
| **Jailer е®үе…Ё** | еҶ…зҪ® jailer е·Ҙе…·жҸҗдҫӣ chroot + йҷҚжқғ + cgroup йҡ”зҰ» |
| **еӨҡзүҲжң¬ж”ҜжҢҒ** | й•ҝжңҹзЁіе®ҡзҡ„зүҲжң¬зӯ–з•ҘпјҢж”ҜжҢҒж»ҡеҠЁеҚҮзә§ |
